# Qcuit Frontend

React/Vite browser companion for the Qcuit library, docs, visualizer, and QML lab.

## Scripts

```bash
npm start       # Vite dev server on :3001, proxies /api to Flask on :5001
npm test        # Vitest manifest checks
npm run build   # Production build to ./build for Vercel
npm run preview # Preview the built app locally
```

The frontend requires Node `>=22.12.0`.
